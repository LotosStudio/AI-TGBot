# AI-TGBot
Как использовать бота, установите его на сервер через ссылку на рипозиторий
https://github.com/LotosStudio/AI-TGBot/edit/main/AI.py

Установите на сервер python3
Ubuntu/Debian
sudo apt install python3

Fedora
sudo dnf install python3

Arch
sudo pacman -S python

sudo apt install python3-pip (для менеджера пакетов pip)
Так же установите необходимые библиотеки через pip
